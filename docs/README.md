## Export Formats

The extension supports two export formats:

1. **Classic Quiz Format** (`v1.0`) - For traditional Canvas quiz question banks
2. **New Quizzes Item Bank Format** (`v2.2`) - For the newer Canvas New Quizzes system

See [EXPORT_SCHEMA.md](./EXPORT_SCHEMA.md) for complete schema documentation.
